# CurioBite — Pre-Launch Landing Page

A single-page pre-launch landing page for [CurioBite](https://algoritmolab.com) with an embedded Tally waitlist form.

## Deploy on GitHub Pages

1. Go to your repo **Settings** > **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Choose **main** branch and **/ (root)** folder
4. Click **Save**
5. Your site will be live at `https://sdipayan.github.io/curiobitetally/`

## Local Preview

Open `index.html` in any browser — no build step required.

## Stack

- Pure HTML + inline CSS (no dependencies)
- Google Fonts (Nunito 900, DM Sans)
- Tally embed SDK for the waitlist form
